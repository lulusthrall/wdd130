Overview

Purpose

The purpose of this website is to serve as a digital portfolio of my personal Pokémon card collection, detailing each card's condition and TCGplayer market value. My goal is to apply HTML and CSS skills to build a clean, visually appealing, and informative gallery about a topic I am passionate about.

Audience

The audience for this site includes other Pokémon collectors, TCG hobbyists looking for value comparisons, and fellow web development students.

Content

Home page

Welcome to the TCG Vault! This site is a personal project showcasing my collection of Pokémon cards, their condition, and their current TCGplayer market value. It was built using HTML and CSS to blend a passion for collecting with new web development skills. Feel free to browse the gallery to see the full collection!

Images for the Home page

The Collection Gallery

This gallery features all the cards in my featured collection. Each card image links to a detailed page with more information, including its set, rarity, condition, and current TCGplayer market value.

Images for the Page 2

The Card Details

This page layout is the template for each individual card. It features a large, high-quality image of the card on one side and a detailed table of information on the other. This data includes the card's name, set, number, rarity, condition, and a direct comparison of its TCGplayer market value.
